# ragtag

Ragtag is a command-line tool for managing tags embedded in plain text note files.
Tags (`@example_tag`) can be used to tag your notes for all sorts of purposes, such as:

* Ideas
* Things to do
* Questions
* ...and more

I keep my notes and track my tasks in plain text markdown files.
This tool aims to be supplement to help me aggregate, organize, query, and manage the tags spread across hundreds of files.
